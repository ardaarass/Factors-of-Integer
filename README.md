Program Hakkında:

Bu program aşağıda belirtilen kurallar dahilinde kullanıcıdan alınan pozitif bir tamsayının istenilen böleninden itibaren pozitif tamsayı bölenlerini ekranda görümtülemektedir.

Programın Çalışma Şartları:

a)Bölenlerinin bulunmasını istediğiniz sayı pozitif bir tamsayı olmalıdır.

b)Hangi bölenden itibaren görüntülenmesini istediğiniz sorulduğunda seçtiğiniz sayı pozitif bir tamsayı olmalıdır.

Nasıl Çalıştırılır?

1)Bölenlerinin bulunmasını istediğiniz sayıyı giriniz.

2)Pozitif bir tamsayı girmemeniz durumunda program sizi uyaracaktır.

3)Sayınızın bölenlerinin hangi değerden itibaren görüntülenmesini istediğinizi giriniz (girdiğiniz sayı görüntülenecek bölenlere dahildir).

4)Girdiğiniz bölen görüntüleme alt sınırı pozitif bir tamsayı değil ise sistem sizi uyaracaktır.

5)Belirttiğiniz şartlara uygun olarak hesaplanan bölenler sırayla listelenecektir.
